git pro读书笔记-Git分支(2)

git分支简介，git的分支模型被称为“必杀技特性”。分支轻量的让人发指（~~~）。git鼓励频繁的使用分支和合并，随着使用的深入，便会理解它的强大和独特。

分支基本操作

  1. 分支创建

  2. 分支切换

  3. 创建并切换到此分支

  4. 分支的合并(把newbranch分支合并到master分支)

  5. 删除分支

  6. 强行删除

分支管理

  1. 查看所有分支（本地）

  2. 查看哪些分支已经被当前分支合并

  3. 查看未合并到当前分支的分支

分支操作步骤

  1. 创建分支并切换到此分支

  2. 在此分支进行修改(本地工作目录)

  3. 添加到缓存区并提交(本地仓库)

  4. 同步到远端仓库

  5. 如果需要把testing分支合并到master分支

  6. 如果testing分支开发完毕，删除

  7. 删除远端仓库tesing分支
